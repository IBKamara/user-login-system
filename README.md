# User Login and Registration System

A full-stack PHP/MySQL login system with user and admin pages.

## Setup Instructions

1. Import `database.sql` to MySQL
2. Rename `config-sample.php` to `config.php`
3. Update database credentials in `config.php`
4. Access the application at `http://localhost/your-folder/register.php`

## Default Admin Credentials
- Username: admin
- Password: [Create your own during setup]
